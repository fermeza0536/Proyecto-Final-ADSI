# Proyecto Final ADSI

Aplicacion para programar cafes antes de entrar a clase.

La aplicacion debe permitir:

- Seleccionar el menu.
- Elegir un pedido.
- Escoger una opcion de pago entre 3 alternativas.
- Confirmar la hora de entrega.
